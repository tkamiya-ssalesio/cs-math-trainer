# CS Math Trainer | 情報I 計算トレーニングアプリ

高校の「情報I」の授業で学習する、**進数計算**やデジタルデータに関する計算のトレーニングを行うためのWebアプリケーションです。インストールや複雑な設定は一切不要で、Webブラウザさえあれば即座に実行・学習を開始できます。

---

## 🚀 特徴

- **豊富な進数変換モード**
  - 2進数 ➔ 10進数 
  - 10進数 ➔ 2進数 (ビットを直接ON/OFFできる**インタラクティブなトグルスイッチUI**を搭載)
  - 10進数 ➔ 16進数
  - 16進数 ➔ 10進数
  - 2進数 ➔ 16進数
  - 16進数 ➔ 2進数 (ビットを直接ON/OFFできる**インタラクティブなトグルスイッチUI**を搭載)
- **3つの難易度設定**
  - **初級 (4-bit / 0〜15)**: 初めて進数計算を学ぶ生徒向け。
  - **中級 (8-bit / 16〜255)**: 標準的な計算練習用。1バイトの構成を理解するのに最適。
  - **上級 (16-bit / 256〜65535)**: より複雑な計算に挑戦したい生徒向け。
- **ゲーム性を持たせた学習体験**
  - 各問題15秒の時間制限があり、素早い計算力を養います。
  - 連続正解で「コンボ」が発生し、獲得スコアがアップします。
  - 終了後は間違えた問題が一覧で表示され、その場で復習できます。
- **授業への導入が非常に簡単**
  - HTML/CSS/JavaScriptのみで構築されているため、ビルド不要でGitHub Pages等に配置するだけで、生徒の端末（Chromebook、iPad、Windows/Mac）からアクセス可能です。

## 🛠️ 技術スタック

- **Structure**: HTML5 (セマンティックタグ)
- **Styling**: Vanilla CSS (CSSカスタムプロパティ、グラスモルフィズム、フルレスポンシブデザイン)
- **Logic**: Vanilla JavaScript (ES6)

## 📦 使い方・動かし方

1. このリポジトリをダウンロード、またはクローンします。
   ```bash
   git clone https://github.com/tkamiya-ssalesio/cs-math-trainer.git
   ```
2. クローンしたディレクトリ内の `index.html` をダブルクリックするか、ブラウザにドラッグ＆ドロップして開きます。
3. ローカルサーバーを起動したい場合は、VS Codeの「Live Server」拡張機能などを使用するとスムーズに起動できます。

## 📈 今後のアップデート予定

情報の授業のカリキュラム進度に合わせて、以下の計算トレーニングモードを追加予定です。
- **文字データの量計算**（ASCIIコード、Unicode等のバイト計算）
- **音声データの量計算**（サンプリング周波数、量子化ビット数、モノラル/ステレオによるサイズ算出）
- **画像・映像データの量計算**（画素数、解像度、フレームレート、色深度からのファイルサイズ算出）
- **圧縮率の計算**（圧縮前後のデータ比較）
